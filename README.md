# chip-8-emu
A CHIP-8 emulator in C++

## Build & Run
Requirements
```bash
sudo apt install cmake libsdl2-dev
```
Compile
```bash
mkdir build
cd build
cmake ..
make
```
Run
```bash
./chip8 [ROM]
```
